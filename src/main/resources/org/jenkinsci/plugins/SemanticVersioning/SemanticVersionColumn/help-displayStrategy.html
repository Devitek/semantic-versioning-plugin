Allows customization of what is shown in the Semantic Version column when the most recent build was unsuccessful. Options are:
  • Show Last Jenkins Build Number: The Jenkins build number of the last build is displayed;
  • Show Last Successful Jenkins Build Number: The Jenkins build number of the last successful build is displayed;
  • Show Last Successful Semantic Version: The Semantic Version of the last successful build is displayed;
  • Show 'N/A': Shows the literal string 'N/A';
